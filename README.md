# screen_shoter
This program writen in C that will screenshot your screen into Downloads folder and send it via webhook you set every 2 seconds. On linux there is some extra setup because it program for screenshoting isnt builded in.
For installation you will need to compile with libcurl like this:
```shell
gcc screen.c -o screen -lcurl 
```
